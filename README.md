# A-Review-of-Cardiac-MRI-Reconstruction-Techniques
Research Project at University, TU Delft, Q5, Individual Project

# Repository Contents

- `report.pdf`: project report

# Summary
Reviewed traditional and emerging cardiac MRI reconstruction techniques, including parallel imaging, compressed sensing, deep learning, and hybrid approaches. Conducted a comparative analysis of these methods and assessed their strengths, limitations, and applicability to cardiac cine MRI. This laid the foundation for the background and literature review sections of my Master's thesis, Deep Learning Guided Reconstruction for Cardiac Cine MRI.
